# 高尔夫系列三：短杆与沙坑 Shot List

## 使用说明

这不是正文文案，而是下一批小黑高尔夫系列的配图规划稿。

目标：

- 延续前两组内容的教学语气
- 每张图只讲一个核心判断
- 优先选择短杆、沙坑、救球和练习反馈这些更适合视觉化的内容
- 保持小黑正文配图体系：白底、少字、手绘、怪诞但清爽

---

## 01 切杆先看落点，不是直接瞄洞

- 放在哪段后：
  适合放在“短切靠落点控制，不靠硬喂到洞边”的段落后。
- 图的主题：
  落点先于洞杯
- 核心意思：
  切杆不是把球直接送到洞，而是先决定落地的小区域，再让球滚过去。
- 结构类型：
  地图路线
- 小黑在做什么：
  小黑拿着一根小旗子，先插在果岭上一小块落点区，球沿着橙色弯线再慢慢滚向洞杯。
- 建议元素：
  果岭坡面 / 落点小岛 / 洞杯 / 橙色滚动路径
- 建议中文标注词：
  `先落这` / `再滚过去` / `别直冲洞`

## 02 沙坑不是先打球，是先用沙

- 放在哪段后：
  适合放在“杆头先切沙，借沙把球带出来”的段落后。
- 图的主题：
  用沙托球
- 核心意思：
  沙坑击球的第一接触对象不是球，而是球下那一层沙。
- 结构类型：
  系统局部
- 小黑在做什么：
  小黑拿着一把夸张的小铲子，从球下方铲起一片沙，沙像传送带一样把球托出坑沿。
- 建议元素：
  沙坑剖面 / 球下沙层 / 杆头切入点 / 出坑轨迹
- 建议中文标注词：
  `先切沙` / `别先碰球` / `沙把球带走`

## 03 救球先回安全区，不要硬当英雄

- 放在哪段后：
  适合放在“恢复球优先回到可打位置”的段落后。
- 图的主题：
  先救活局面
- 核心意思：
  球进树林或长草后，正确决定往往不是硬穿，而是先把局面带回可继续操作的位置。
- 结构类型：
  前后对比
- 小黑在做什么：
  左边的小黑正试图把球硬塞过一排树，右边的小黑把球推回一块干净平地，脸一样严肃但结果完全不同。
- 建议元素：
  树丛 / 长草 / 狭窄缝隙 / 安全落点区 / 红叉与橙箭头
- 建议中文标注词：
  `别逞强` / `先回正轨` / `下一杆更重要`

## 04 球位一变，弹道就变

- 放在哪段后：
  适合放在“球位影响击球时机和轨迹”的段落后。
- 图的主题：
  同一个人，不同相遇时机
- 核心意思：
  球放前一点或后一点，击球点和飞行轨迹就会明显变化。
- 结构类型：
  前后对比
- 小黑在做什么：
  两个小黑站在同一条挥杆弧线旁，一个把球放后，打出低弹道；另一个把球放前，打出高弹道。
- 建议元素：
  挥杆弧线 / 两个球位点 / 高低两条球路 / 站位基准线
- 建议中文标注词：
  `靠后更压` / `靠前更高` / `时机变了`

## 05 练习不要只打球，要看反馈闭环

- 放在哪段后：
  适合放在“练习时要知道每一组球在验证什么”的段落后。
- 图的主题：
  不是堆球，是回收反馈
- 核心意思：
  练习真正有效的地方，不在于打一大堆球，而在于每打一组就回看结果、再调整下一组。
- 结构类型：
  Workflow 流程
- 小黑在做什么：
  小黑把一篮球送进练习台，另一侧不是单纯出球，而是掉出三张小纸条：落点、方向、触球质量；小黑再把纸条塞回一个调整旋钮。
- 建议元素：
  球篮 / 练习台 / 反馈纸条 / 调整旋钮 / 回流箭头
- 建议中文标注词：
  `打完要看` / `方向` / `落点` / `触球` / `再调一轮`
