# Golf Xiaohei Next Series Prompt

Use `$ian-xiaohei-illustrations` to continue the golf teaching series.

## Planning Prompt

```text
Use $ian-xiaohei-illustrations 先不要生图。
请继续这个高尔夫教学系列，先输出 5 张左右的 shot list。

要求：
- 主题围绕：短杆 / 沙坑 / 球场策略 / 练习习惯 / 上场决策，任选其一做成一组
- 每张图只讲一个核心教学点
- 保持“小黑怪诞正文配图”风格
- 不要平均配图，只挑最值得视觉化的认知锚点

每张图写清楚：
- 放在哪段后
- 图的主题
- 核心意思
- 结构类型
- 小黑在做什么
- 建议元素
- 建议中文标注词
```

## Generation Prompt

```text
Use $ian-xiaohei-illustrations 把下面这组高尔夫教学内容生成 4 张小黑怪诞正文配图。

要求：
- 16:9 横版
- 纯白背景
- 黑色手绘线稿
- 少量红橙蓝中文手写批注
- 每张图只讲一个教学点
- 不要做 PPT 信息图
- 不要可爱卡通
- 小黑必须承担核心动作
- 不要左上角标题

优先挑这些结构：
- 击球顺序
- 常见误区前后对比
- 球路与落点关系
- 上场决策路径
- 练习反馈闭环
```

## Single-Image Prompt Seed

```text
Use $ian-xiaohei-illustrations 为这个高尔夫观点生成一张正文配图：

“切杆先选落点，不要只盯着洞杯。”

画面要怪诞但清爽。
小黑必须承担核心动作。
中文标注最多 5 个，尽量短。
不要复刻旧案例。
```
