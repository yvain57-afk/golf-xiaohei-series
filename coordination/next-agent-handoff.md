# Next Agent Handoff

## Goal

Continue the golf xiaohei series in a way that another agent can pick up cleanly.

## Do First

1. Read `coordination/series-status.md`.
2. Read `prompts/golf-xiaohei-next-series.md`.
3. Inspect existing assets in `outputs/` to avoid duplicating current topics.

## Constraints

- Treat this repo as an output repo, not an application repo.
- Do not refactor structure unless the user asks.
- Do not overwrite existing output files by default.
- Keep diffs small and reviewable.

## Style Guardrails

- Pure white background.
- Black hand-drawn line art.
- Sparse red/orange/blue handwritten Chinese labels.
- One image explains one teaching point.
- Xiaohei must perform the core action, not stand as decoration.
- Avoid PPT, diagram, courseware, or cute mascot vibes.

## Good Next Directions

- Add a new series batch for short game, bunker play, course strategy, or practice habits.
- Turn one existing markdown article into a shot list first, then generate image prompts.
- Improve naming and indexing only if it helps future continuation.

## Avoid

- Recreating the exact same composition from prior examples.
- Stuffing multiple golf concepts into one image.
- Adding long explanatory text inside images.
